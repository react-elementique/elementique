# Input

Input component with different props.

### Primary
___

<Canvas of={Input.Primary} />

## Usage

```ts
import {Input} from "elementique";

const Example = () => {
  return (
      <Input placeholder="Enter text" variant={"primary"} />
  );
};

export default Example;
```

### Float
___

<Canvas of={Input.Float} />

## Usage

**Note**: You need to wrap the float input inside a div to handle it's width.

```ts
import {Input} from "elementique";

const Example = () => {
  return (
    <div style={{ width: "400px" }}>
      <Input placeholder="Enter text" variant={"float"} />
    </div>
  );
};

export default Example;
```
___

#### Arguments

- **variant** - <em>primary</em> or <em>float</em>
- **name** - This value will be passed to the id and name tags of input
- **type** - This prop accepts the type of input ex: <em>text</em>, <em>email</em>
- **width** - Custom width of the input
- **maxWidth** - Maximum width of the input
- **p** - Custom padding for the input as number
- **placeholder** - This props accepts the prop to be displayed as the placeholder for the input
- **setValue** - This prop is for the React useState <em>setValue</em> function to change the <em>value</em>
- **value** - The value entered inside the input can be accessed using this prop
- **border** - Custom border for the input box
- **fontsProperty** - Custom fontsProperty is to be entered here in the form of an object: \{ size: string, weight: number, family: string \}
