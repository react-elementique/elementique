# Input

Input component with different props.

#### Example

<Canvas of={Input.Primary} />

## Usage

```ts
import {Input} from "sld-ui";

const Example = () => {
  return (
      <Input
        type="text"
        onClick={() => {}}
        disabled={false}
        placeholder="Enter text"
        size="small"
        primary
      />
  );
};

export default Example;
```

#### Arguments

- **text** _`() => void`_ - A string that represents the text content of the Input.
- **primary** - A boolean indicating whether the Input should have a primary styling or not. Typically, a primary Input stands out as the main action in a user interface.
- **disabled** - A boolean indicating whether the Input should be disabled or not. When disabled, the Input cannot be clicked or interacted with.
- **size** - A string with one of three possible values: "small," "medium," or "large." It defines the size or dimensions of the Input.
- **onClick** - A function that is called when the Input is clicked. It receives a MouseEventHandler for handling the click event on the Input element.
